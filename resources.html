Resources

Learning to code websites is a difficult but rewarding goal.Having a few solid resources for learning material is crucial to stay on top of this constantly changing technology.

Below there are a few sites I used in the few months as a web development student.I believe these sites are great resources of knowledge for web development.From these kind of sites I think you will be able to learn a wide variety of new skills via online videos, code exercises and tests.

Lynda

Probably this is one of the most popular online tutorial libraries today.It contains s lot of web development, web design and many other courses.

Coursera

Whether you want to master a specific career skill or you just want to enrich your knowledge about web development Coursera is the perfect place.Every course on Coursera is taught by top instructors from the world`s best universities and educational institutions.

These websites are great when you are looking to sit down and dig into something new.If you are knee-deep in a projectand you are stuck on a bug or you just don`t know how to proceed these sites will not help you much.Some of the best sites I use to find answers for development-related issues are listed below:

Stack Overflow

Stack Overflow is an amazing resource for developers to post issues they are having and then get answers from others in the development community wo had either tackled a similar issue or have an idea to solve the problem

Mozilla developer Network

Mozilla Developer Network is a comprehensive developer resource on the internet, especially when it comes to common front-end technologies such as HTML, CSS and JavaScript.You can learn web development from nothing.
